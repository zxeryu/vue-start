# Triage labels

## 标签词汇

| 角色 | 标签 | 描述 |
|------|-------|-------------|
| needs-triage | `needs-triage` | 维护者需要评估 |
| needs-info | `needs-info` | 等待报告者提供信息 |
| ready-for-agent | `ready-for-agent` | 规格完备，agent 可接手 |
| ready-for-human | `ready-for-human` | 需要人类实现 |
| wontfix | `wontfix` | 不会处理 |

无自定义映射 — 使用默认名称。
